📖 Universal SDK & API Embedding Guide

🚀 Universal 1-Line SDK Embed

Include sdk.js in the <head> of any website (Zero inline CSS added, fully customizable):

<script src="https://pqrstu34.github.io/test/sdk.js"></script>

💡 Embed Any File Using Universal ia-id="<ID>"

Internet Archive automatically detects the file format. Simply use ia-id="<ID>" on any element:

1. Embed File / Media via ID:
<div ia-id="dd4ef44a"></div>
2. Embed Download Link via ID:
<a ia-id="b2e414f6">Download File</a>
3. Direct URL Parameter API Query:
https://pqrstu34.github.io/test/?crc32=dd4ef44a
4. JavaScript Helper Functions:
const file = getFileByCrc32('dd4ef44a');
console.log(file.download_url, file.format, file.size);
